# gabigabi

png / jpg 画像をガビガビにするツールです。

ドット絵の資料画像作成のために開発しました。

ブラウザまたは CLI として利用することができます。

- ブラウザ版: <https://zztkm.github.io/gabigabi/>
- CLI 版: `go install github.com/zztkm/gabigabi@latest`

## CLI Usage

```bash
gabigabi -in input.jpg -out output.png
```
